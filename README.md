# 📦 Java Inventory Management System

Java Inventory Management System is a desktop application developed in Java using JavaFX and FXML.

The project simulates a small inventory and sales management system where users can browse products, add items to a shopping cart, validate stock availability and complete purchases through a graphical interface.
